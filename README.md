# CrackIt! - Website

This repository contains the landing page and website for **CrackIt!**, the ultimate AI coding sidekick.

## Overview
The website is a simple, hacker-themed terminal interface that allows users to download the `CrackIt_11.exe` executable directly. 

## Files
- `index.html`: The main landing page.
- `style.css`: The stylesheet featuring a "hacker terminal" aesthetic and matrix canvas overlay.
- `script.js`: Contains the logic for the matrix rain animation and YouTube iframe controls.
- `CrackIt_11.exe`: The compiled Windows executable of the application.

## Development
To view or modify this website locally, simply open `index.html` in any modern web browser.
